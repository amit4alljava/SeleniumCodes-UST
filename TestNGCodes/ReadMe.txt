TestNG is a testing framework which is used for different range of testing. 
e.g. Unit Testing, Integration Testing.

TestNG is a testing framework inspired from JUnit and NUnit, but introducing 
some new functionalities that make it more powerful and easier to use.

http://mvnrepository.com/artifact/org.testng/testng/6.9.6
